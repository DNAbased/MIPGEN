Welcome.

MIPgen is a fast, simple way to generate designs for MIP assays targeting hundreds or thousands of genomic loci in parallel.

Packaged with MIPgen are scripts that aid in visualization of MIP designs and processing of MIP sequence reads to SAM files that can then be passed through any standard variant calling pipeline. See the GitHub page README for more details.

MIPgen is provided freely for non-commercial use. Contact Jay Shendure (shendure [at] uw.edu) for commercial application opportunities.
